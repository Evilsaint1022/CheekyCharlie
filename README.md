# üéÆ Application Commands

**Available for Everyone:**

- `/avatar` ‚Äî View your's or someone elses avatar
- `/ping` ‚Äî Check Bots Latency
- `/cat` ‚Äî Generates a random picture of a cat
- `/dog` ‚Äî Generates a random picture of a dog
- 
- `/github` ‚Äî Github link to CheeckyCharlies Repo
- `/invite` ‚Äî Generates a random invite link
- 
- `/level` ‚Äî Check Current Level
- 
- `/deposit` ‚Äî Deposit Currency into your Bank
- `/withdraw` ‚Äî Withdraw Currency from your Bank
- `/balance` ‚Äî Check your balance
- `/leaderboard` ‚Äî Balance Leaderboard
- `/daily` ‚Äî Daily Currency Collect
- `/pick` - Picks coins when the DropParty drops`

**Staff-Only Commands:** (`Administrator permissions required`)

- `/echo` ‚Äî Echo Messages
- 
- `/set-whitelisted-roles` ‚Äî Set whitelisted Roles.
- `/remove-whitelisted-roles` ‚Äî Remove whitelisted Roles.
- 
- `/set-level-channel` ‚Äî Set Level Notification Channel
- `/remove-level-channel` ‚Äî Remove Level Notifcation Channel
- 
- `/set-drop-party-channel` ‚Äî Set Drop Party Channel
- `/remove-drop-party-channel` ‚Äî Remove Drop Party Channel
- 
- `/starboard-set-channel` - Sets Starboard Channel
- `/starboard-set-count` - Sets Amount of emojis needed for Starboard
- `/starboard-set-emoji` - Sets sets Starboard Emoji
